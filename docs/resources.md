# 资料清单

## AIOps 与智能运维

- [Red Hat: The journey from observability to AIOps automation](https://www.redhat.com/en/topics/automation/observability-to-aiops-automation)
  AIOps、可观测性、事件驱动自动化的整体框架。
- [SREWorks](https://sreworks.cn/)
  中文开源智能运维平台参考，覆盖 DataOps、AIOps、多云管理、运维应用工厂。

## SRE

- [Google SRE Books](https://sre.google/books/)
  SRE 原始教材和实践手册，重点看 SLO、toil、monitoring、incident response。
- [Google Cloud SRE](https://cloud.google.com/sre)
  用于理解 SRE 的岗位、实践和组织落地。

## 可观测性

- [OpenTelemetry](https://opentelemetry.io/)
  统一遥测标准，重点看 metrics、logs、traces、collector。
- [Prometheus overview](https://prometheus.io/docs/introduction/overview/)
  监控和告警基础。
- [Grafana documentation](https://grafana.com/docs/)
  仪表盘、告警、日志和可观测性平台实践。

## 云原生

- [Kubernetes overview](https://kubernetes.io/docs/concepts/overview/)
  容器编排、自动恢复、扩缩容和发布策略。
- [CNCF Landscape](https://landscape.cncf.io/)
  了解云原生生态，不要试图一次学完。

## 数据分析与异常检测

- [scikit-learn IsolationForest](https://scikit-learn.org/stable/modules/generated/sklearn.ensemble.IsolationForest.html)
  第一个异常检测项目的模型参考。
- [scikit-learn outlier detection](https://scikit-learn.org/stable/modules/outlier_detection.html)
  了解 novelty detection 和 outlier detection 的区别。
