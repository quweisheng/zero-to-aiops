## 天津现实策略

天津本地 AIOps 岗位数量可能少于北京、上海、深圳，所以采用三层策略：

1. 天津本地：优先投 Lenovo、金融、制造、云服务、政企信息化、工业互联网相关岗位。
2. 北京远程/混合：搜索北京 SRE / DevOps / AIOps，筛选可远程或低频到岗。
3. 全国远程：投递云原生、可观测性、监控平台、自动化运维方向。
