# AIOps 工程师能力地图

## 基础运维

- Linux：进程、文件系统、网络、systemd、性能分析。
- 网络：DNS、HTTP、TLS、负载均衡、常见故障定位。
- 数据库与中间件：MySQL、Redis、MQ 的基础监控和排障。
- 自动化：Shell、Python、Ansible 或其他运维编排工具。

## SRE 与可靠性

- SLI/SLO/SLA。
- 错误预算。
- 监控设计和告警治理。
- 事件响应和复盘。
- 容量评估和性能压测。

## 可观测性

- Metrics：Prometheus、PromQL、exporter。
- Logs：结构化日志、日志模板、错误聚合。
- Traces：OpenTelemetry、链路上下文、跨服务定位。
- Dashboards：Grafana、业务视图、值班视图。

## 云原生

- Docker：镜像、容器、网络、存储。
- Kubernetes：Pod、Deployment、Service、Ingress、ConfigMap、Secret、HPA。
- CI/CD：流水线、发布策略、回滚、灰度。
- 平台工程：把重复流程做成平台能力。

## 数据与 AI

- Python 数据处理：pandas、numpy、可视化。
- 异常检测：阈值、动态基线、IsolationForest。
- 事件关联：时间窗口、拓扑、相似日志、变更记录。
- LLM 应用：告警摘要、runbook 推荐、知识库检索。
- 评估：误报、漏报、MTTD、MTTR、人工节省时间。

## 求职表达

- 不是说“我学过 AIOps”，而是说“我用可观测数据做了一个能降低告警噪声的工具”。
- 不是说“我懂 AI”，而是说“我知道模型何时只能推荐，何时可以进入自动化流程”。
- 不是说“我年龄 37”，而是说“我有生产经验，知道自动化必须带审计、分级和回滚”。
